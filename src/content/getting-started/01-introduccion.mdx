# Introducción

**GIMA (Gestión Integral de Mantenimiento y Activos)** es un ecosistema de aplicaciones diseñado para modernizar y optimizar las operaciones de mantenimiento industrial, con un enfoque inicial en el contexto organizacional de la **UNEG**.

El sistema integra capacidades avanzadas de **Inteligencia Artificial** para facilitar la captura de datos, la generación de reportes y la toma de decisiones basada en activos.

## 🎯 Objetivos del Proyecto

1. **Unificar la Gestión:** Centralizar el control de activos, inventario y órdenes de trabajo en una única plataforma accesible.
2. **Potenciar con IA:** Utilizar modelos de lenguaje (LLMs) y visión computacional para automatizar tareas repetitivas de entrada de datos.
3. **Mejorar la Disponibilidad:** Optimizar los tiempos de reparación (MTTR) y los intervalos entre fallos (MTBF) mediante un seguimiento riguroso.
4. **Onboarding Intuitivo:** Proporcionar una interfaz moderna y guías claras para técnicos y administradores.

## 👥 Stakeholders y Usuarios

- **Administradores de Sistema:** Gestión de usuarios, roles y configuraciones globales.
- **Gerentes de Mantenimiento:** Supervisión de KPIs, aprobación de presupuestos y reportes ejecutivos.
- **Técnicos de Campo:** Ejecución de órdenes de trabajo, reporte de hallazgos y uso de herramientas de voz/visión.
- **Analistas de Inventario:** Control de stock, gestión de proveedores y puntos de reorden.

## 🏗️ Alcance Técnico

El ecosistema se divide en tres pilares fundamentales:

- **Backend (Laravel):** El motor principal, encargado de la lógica de negocio, base de datos y API REST.
- **AI Chatbot (Next.js):** Interfaz conversacional inteligente con capacidades multimodales.
- **Frontend (Next.js):** Aplicación web administrativa para la gestión integral.
